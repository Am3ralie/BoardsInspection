# BoardsInspection
Система автоматизации учета комплектующих и фиксации результатов технических испытаний. 
Разработано для оптимизации складского учета и перевода бумажной отчетности в цифровой формат.

## Основной функционал
Учет комплектующих: Ведение базы данных компьютерных плат и других компонентов.
Контроль испытаний: Модуль для фиксации результатов технических проверок.
Сетевое взаимодействие: Клиент-серверная архитектура для одновременной работы нескольких пользователей.
Автоматизация отчетности: Генерация данных для контроля качества и инвентаризации.

## Технологический стек
Язык программирования: C# (.NET Framework/Core)
База данных: SQL (проектирование архитектуры таблиц, оптимизация запросов)
Архитектура: WinForms (Desktop) для обеспечения стабильной работы в локальной сети предприятия.

## Назвначение
ПО позволяет заменить ручной ввод данных в журналы, исключает потерю информации об испытаниях и ускоряет поиск нужной продукции.
